A server for the TRNML e-ink dashbaord.

## Alpine Linux Cheatsheet

See running services:

    sudo rc-status

Start the trmnl_srv service:

    sudo rc-service trmnl_srv start

Stop the trmnl_srv service:

    sudo rc-service trmnl_srv stop
